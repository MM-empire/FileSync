# File Sync for terminal

## Command list

```
filesync [-h | --help] [--sync-list=<path>] [--config=<path>]
    add <path-to-file> <path-to-file>
    <!-- [-D | --directories] -->
    <!-- add <path-to-file> <path-to-file> -->
    <!-- filesync hello.file /home/mikhail/Documents -->
    <!-- delete <path> -->
    <!-- diff <path> -->
    sync <path> [--all] [-C | --current-dir]
    update <path> [--all]
    list [--changed | -c] [-C | --current-dir]
```
